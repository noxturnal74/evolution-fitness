# Evolution Fitness Malang Landing Page

Website statis company profile untuk Evolution Fitness Malang dengan nuansa premium, parallax hero, section program, jadwal kelas, membership, personal training, CTA WhatsApp, galeri visual, dan embed Google Maps.

## File

- `index.html` - struktur landing page, SEO meta, schema bisnis, CTA, dan embed Google Maps.
- `styles.css` - visual system responsif, layout, parallax-friendly sections, pricing, schedule, dan mobile menu.
- `script.js` - navigation state, scroll reveal, counter, dan efek parallax ringan.

## Preview Lokal

Buka `index.html` langsung di browser, atau jalankan server statis:

```bash
python -m http.server 4173
```

Lalu buka `http://localhost:4173`.

## Deploy GitHub Pages

1. Push repository ke GitHub.
2. Buka repository Settings.
3. Masuk ke Pages.
4. Pilih deploy dari branch `main` folder `/root`.
5. Simpan, lalu tunggu URL GitHub Pages aktif.
